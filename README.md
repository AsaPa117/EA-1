Deliverable-Specific Content:
The link below is a repository for a project that I've done at Hackathon. Its a fitness application that determines user best type of workout.
link: https://github.com/AsaPa117/Hackathon.git
Description: The fitness application is written in Python and uses a machine learning algorithm, k-means clustering, as its core. In the project, we have only two main files: FitnessPlanUI and FitnessPlan. In the FitnessPlanUI, it creates a window where the user can interact with the program. The window displays questions the user must answer and, using the given data, prints a result. However, before the program returns a result, it must process the user's data in another file, fitnessPlan. The fitnessPlan file uses an imported function to create a machine learning algorithm. It takes in generic samples of categorial data and plots them into a clustered graph. When the ML begins to show visible clusters, it can produce accurate results and is able to process user data. Once done, it sends the result to the UI file and, depending on the result, displays the user's fitness goal. 
Running the Program: If you want to test the application, you can clone the link on a local computer. However, before you can run the program, you might need to download and import libraries for KModes ML, PySide, and Python from the terminal. Once downloaded, you can use Visual Studio Code to start it off.
